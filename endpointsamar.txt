Agencyzoom API (1.0.0)
Download OpenAPI specification:Download

E-mail: support@agencyzoom.com
License: Propriety license
Terms of Service
Agencyzoom API enables third party applications to integrate with Agencyzoom to manage lead, customer, policy, and tasks. The management operations include actions such as creation, retrieval, updating, and deletion.

Currently, Agencyzoom has a rate limit of 30 calls per minute during the day, 60 calls per minutes from 10PM CT to 4AM CT, as of Nov 21, 2023.

Getting Started
Using authentication endpoints as described below, the integrator can function on hehalf of the logged in user. The login endpoint returns a JWT token. Subsequent calls need pass the token as HTTP header bearer token.

Authorization: Bearer <token>
Please note, the permissions afforded to the caller are the same as those for the logged in user. Therefore, the user name used for the integration has to be the agency owner if the integrated applications would like to access all the functionalities.

Versioning
Agencyzoom API is backward compatible. Currently there is only one version.
